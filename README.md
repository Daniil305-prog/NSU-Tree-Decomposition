# Вычисление древовидной декомпозиции графа (Treewidth Solver)

Данный проект разработан в рамках учебного курса НГУ. Программа реализует 5 различных эвристик для поиска оптимального порядка исключения вершин графа с целью минимизации ширины его древовидной декомпозиции (treewidth).

## Реализованные эвристики
1. **Minimum Degree** — жадный выбор вершины с минимальной текущей степенью.
2. **Minimum Fill-In** — выбор вершины, минимизирующей количество добавляемых хорд (edges).
3. **Maximum Cardinality Search (MCS)** — последовательный выбор вершин с максимальным числом посещенных соседей.
4. **Nested Dissection** — алгоритм вложенных сечений на основе поиска сепараторов (разделяй и властвуй).
5. **Гибридный алгоритм (ND + MCS)** — комбинация глобального разделения и локального упорядочивания.

## Особенности реализации
* **Встроенный валидатор:** Автоматически проверяет построенное дерево на строгое соответствие трем математическим аксиомам древовидной декомпозиции.
* **Информативный парсер:** Безопасное чтение графов в стандартном формате PACE/DIMACS (`.gr`) с защитой от ошибок в структуре файла и выходом за границы памяти.
* **Оптимизация по памяти:** Использование динамически расширяемых списков смежности на чистом C.
